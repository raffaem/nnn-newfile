# newfile

A nnn (=nÂ³) plugin to create new file.

Use templates in `$HOME/Templates/` and `fzf` to choose among them.

# Installation

1. Run `install.sh`.
2. Map to a keybinding by exporting
    ```
    NNN_PLUG='n:newfile;'
    ```
